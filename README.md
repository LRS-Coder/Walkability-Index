# Walkability-Index

## Install and Run Programme

### Clone Repository

```python
git clone https://github.com/LRS-Coder/Walkability-Index.git
cd Walkability-Index
```

### Activate Environment

```python
conda env create -f environment.yml
conda activate walkability
```

### Run Programme

```python
python main.py
```
